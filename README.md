## 先进性登录才可以进入系统
	a. 登录的时候根据账号，名字，密码验证用户信息是否正确
	b. 根据不同的用户权限，系统返回不同的显示界面
## 教师登录
	a. 可以对作业，课程，成绩，消息数据库进行增删改查
	b. 可以修改自己的个人信息
## 管理员登录
	a. 可以对教师，学生数据库进行增删改查
	b. 可以修改自己的个人信息 
## 学生登录
  	a. 可以对作业、消息数据库进行增删改查，可以查看成绩、课程数据库
	b. 可以修改自己的个人信息
## 论坛
	a. 可以进行分享和一些感悟的写作
	b. 附带一个消息的评论功能以此来完成双方的交流
